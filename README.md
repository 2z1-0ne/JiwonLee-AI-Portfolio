## Main Projects

### 1. RNFL-Guided Visual Field Denoising
SS-OCT RNFL 구조 정보를 활용해 24-2 시야검사 데이터의 노이즈를 줄이는 딥러닝 기반 디노이징 모델을 개발했습니다. 데이터 매칭, 전처리, 모델 설계, 성능 평가까지 end-to-end로 수행했습니다.

### 2. Glaucoma Visual Field Pattern Analysis
24-2 및 10-2 시야검사 데이터에서 녹내장 진행 패턴을 분석하기 위한 비지도학습 기반 연구에 참여했습니다. Archetypal Analysis와 Fuzzy C-Means 기반 패턴 분해 및 진행 예측 분석을 수행했습니다.

### 3. BERT-Based Prediction Using Tabular Clinical Data
전자의무데이터 기반 급성 심근경색 예측을 위해 BERT 기반 모델을 정형 데이터에 적용하는 실험을 수행했습니다. 수치형 데이터를 column-wise embedding하여 Transformer 입력으로 활용했습니다.
